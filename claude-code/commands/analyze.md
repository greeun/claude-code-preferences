# Analyze Codebase

코드베이스의 현재 상태를 분석하고 인사이트를 제공해주세요.

**분석 항목:**

1. **코드 메트릭스**
   - 파일 수 및 코드 라인
   - 컴포넌트 수
   - API 엔드포인트 수
   - 복잡도가 높은 파일

2. **아키텍처**
   - 디렉토리 구조 분석
   - 계층 분리 상태
   - 의존성 그래프
   - 중복 코드 패턴

3. **기술 스택**
   - 사용 중인 주요 라이브러리
   - 버전 상태
   - 업데이트 필요 항목

4. **코드 품질**
   - TypeScript 커버리지
   - 테스트 커버리지
   - Linter 경고
   - TODO/FIXME 항목

5. **성능 지표**
   - 번들 크기 분석
   - 큰 의존성 식별
   - 최적화 기회

6. **보안 상태**
   - 의존성 취약점
   - 보안 패턴 준수

**결과물:**
- 📊 통계 요약
- 💪 강점
- ⚠️ 개선 영역
- 🎯 권장 사항
