# Code Cleanup

코드베이스를 정리하고 불필요한 요소를 제거해주세요.

**정리 항목:**

1. **사용하지 않는 코드**
   - 미사용 import 제거
   - Dead code 제거
   - 주석 처리된 코드 삭제
   - 미사용 파일 제거

2. **의존성**
   - 사용하지 않는 패키지 제거
   - 중복 의존성 정리
   - `npm outdated` 확인

3. **코드 스타일**
   - Linter 규칙 적용
   - 일관된 포맷팅
   - 네이밍 컨벤션 통일

4. **파일 구조**
   - 적절한 디렉토리 배치
   - 파일명 일관성
   - 논리적 그룹핑

5. **주석 및 문서**
   - 오래된 주석 업데이트/제거
   - TODO 항목 처리
   - JSDoc 정리

6. **예외 사항**
   - shared/ 이하 코드는 적용 대상에서 제외

**실행:**
1. `npm run lint` 실행
2. ESLint 경고 해결
3. TypeScript 에러 0개 확인
4. 테스트 통과 확인
5. 빌드 성공 확인

**결과:**
- 정리된 항목 요약
- 코드 품질 개선 사항
