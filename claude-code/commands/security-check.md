# Security Check

보안 취약점을 점검하고 개선 방안을 제안해주세요.

**점검 항목:**

1. **인증/인가**
   - JWT 토큰 검증 로직
   - 권한 체크 누락 여부
   - 세션 관리

2. **입력 검증**
   - Zod 스키마 검증 누락
   - SQL Injection 방지 (Prisma)
   - XSS 방지
   - CSRF 토큰

3. **API 보안**
   - Rate limiting 적용
   - 민감 정보 노출 (API 응답)
   - CORS 설정
   - API 키 관리

4. **데이터 보호**
   - 패스워드 해싱
   - 환경 변수 관리
   - 로그 민감정보 마스킹
   - IP 주소 처리

5. **의존성**
   - 취약한 패키지 확인
   - 버전 업데이트 필요 여부

**결과:**
- 발견된 취약점 목록
- 우선순위별 수정 방안
- 보안 강화 권장 사항
